# Automated Detection of Arterial Blood Pressure Signal Integrity Failures

**A Reproducible Research Repository for the IEEE JBHI Submission**

---

## ğŸ“Š Abstract

This study presents the first systematic audit of Arterial Blood Pressure (ABP) signal integrity in open clinical databases. We analyzed 400 five-minute ABP windows from two major databases: **VitalDB** (a surgical ICU database from Korea) and **CHARIS** (a traumatic brain injury ICU database from the USA). Our automated auditing framework, based on established physiological plausibility checks, revealed a staggering **74.2% combined failure rate**. The failure rate was **97.8% in VitalDB** and **25.4% in CHARIS**, highlighting significant, database-specific data quality issues. We further developed and validated a Random Forest-based machine learning model to automate this auditing process, achieving a **ROC-AUC of 0.994** and **PR-AUC of 1.000**. This work establishes that signal integrity auditing is a mandatory prerequisite for the development of trustworthy medical AI and provides a reproducible framework for conducting such audits.

- **High-Performance Automation**: Our ML model can automate the detection of these failures with near-perfect accuracy, enabling scalable quality control.
